Sharpen your tool before the work

Assign roles as custom instructions
How would you like ChatGPT to respond?
Exceptions:
When I start my prompt as NEW, please forgot about your role and start as a brand clean instance. This will be used for all topics not related with my software development job.
When I start my prompt with SHORT, then please give me as concise answer as you can. Skip the commentary and provide very concise anwer it directly answer the question.
When I start my prompt with CODE, then give me just code without any commentary and extra information.
When I start my prompt with WAIT then just acknowledge you got the message by "waiting", as there will be more context. When there is no more context, there will be READY command.
When providing a code, give instructions first
Or divide a message into separate messages ("WAIT")

Concrete use cases

Generate templates
  • AWS CloudFormation/SAM
  • AWS policies
  • OpenApi definitions
Script generation
  • bash / python / awk / any unix tools
  • Following notation is useful:
    Input: XXX, Output: XXX
Regex creation
  • Always paste text fragment to be matched against
Markdown tables
  • Precisely modify table content (e.g. "swap the columns")
  • Create tables using arbitrary notation
Example prompt
Generate markdown table. I use "RXCX: content" notation to indicate a cell, where R is row, C is column and X is number.
Absent cells should be empty.
R1C3: Alice
R2C4: Bob
R5C1: fooBar

  • Programming language conversions. Also from pseudo-language.
