Prepare key commands to tailor responses
"When the message starts with NEW, forgot about your current role and start as a brand clean instance."
"When the message starts with SHORT, give the most concise answer answering directly to the question and skip the commentary."
"When the message starts with CODE, provide only code without any commentary and context."
"When the message starts with WAIT just acknowledge you got the message by "waiting", as there will be more context. When there is no more context, message will start with READY command."
